# SlowVersion – исходная версия проекта

## Описание

Данный проект представляет собой консольное приложение на языке C#, 
которое вычисляет факториал чисел, введённых пользователем через консоль.

Программа принимает строку чисел, разделённых пробелами, 
преобразует их в список и поочерёдно вычисляет факториал каждого числа.

## Особенности реализации

- Используется консольный ввод и вывод
- Применяются строки, массивы и коллекции
- Вычисления выполняются последовательно
- Кэширование отсутствует
- Асинхронность не используется
- Вся логика сосредоточена в одном классе Program

## Выявленные проблемы

В ходе анализа были обнаружены следующие недостатки:

1. Повторные вычисления одинаковых значений
2. Отсутствие параллельной обработки
3. Отсутствие кэширования результатов
4. Нарушение принципов объектно-ориентированного программирования
5. Низкая масштабируемость при увеличении количества данных
